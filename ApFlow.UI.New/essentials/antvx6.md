# 2. DAG 图分支汇聚时的经典层级计算问题。
问题根源：你的 BFS 算法在遍历时，第一个访问到合并节点的路径就决定了它的层级，导致另一条路径的层级被"切断"。

正确布局                     错误布局（你的代码）
<pre>
    A                          A
  ┌─┴─┐                      ┌─┴─┐
  B   C                      B   C
  │   │                      │   │
  │   D ←───── 错误：D 与 C 平齐
  └───┘
    D ←───── 正确：D 应在 C 下方
</pre>
解决方案：基于最长路径的层级计算
合并节点的层级应该基于最长路径，而不是第一个访问的路径。
