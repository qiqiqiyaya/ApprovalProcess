# 项目架构

## 核心算法模型
- **图（Graph）**：有向无环图结构
- **顶点（Node）**：图中的节点元素
- **边（Flow）**：连接节点的流向关系

## 前端技术栈

| 层级      | 技术选型      | 版本   | 说明                       |
| :-------- | :------------ | :----- | :------------------------- |
| 基础框架  | Angular       | 21.0.0 | 现代前端开发框架           |
| UI 组件库 | ng-zorro-antd | 21.0.2 | Ant Design 的 Angular 实现 |
| 图形引擎  | @antv/x6      | -      | 专业级图编辑引擎           |
| 表单方案  | @delon/form   | -      | 动态表单构建工具           |

## 设计原则

1. **可视化核心** — 基于 @antv/x6 实现图（Graph）的渲染与交互，支持有向无环图（DAG）结构，顶点为 Node，连接线为 Flow

2. **组件标准化** — 全面采用 ng-zorro-antd 组件库，确保界面风格统一

3. **设计规范性** — 严格遵循 Ant Design 设计规范，保证用户体验一致性
