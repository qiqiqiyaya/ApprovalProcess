使用说明：

  • 拖拽左侧节点到画布添加节点
  • 点击左侧节点在画布中心添加节点
  • 拖拽节点端口进行连接
  • 点击节点进行选中
  • 使用工具栏按钮进行布局调整
  • 并行节点和合并节点会自动垂直对齐
  • Shift + 鼠标拖拽平移画布
  • 鼠标滚轮缩放画布
  • Delete/Backspace删除选中元素
  • Ctrl+A / Meta+A全选所有元素
  • Ctrl+Z / Meta+Z撤销操作
  • Ctrl+Shift+Z / Meta+Shift+Z重做操作
  • + / NumPad+放大画布
  • - / NumPad-缩小画布
